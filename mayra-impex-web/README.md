# Mayra Impex Web Admin

This is the admin-only dashboard for Mayra Impex. Only admins can log in and manage products and other resources. There is no client-facing frontend; all customer interactions happen via the mobile app.

## Features

- Admin login
- Admin dashboard (extendable)
- Product/resource management (to be implemented)

## Getting Started

1. Install dependencies:
   ```
   cd mayra-impex-web
   npm install
   ```
2. Run the development server:
   ```
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Notes

- This web dashboard is for admin use only.
- All product displays and client features are on the mobile app.
